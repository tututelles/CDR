⚙️ Configurar banco de dados Supabase
Conecte para sincronizar com todos os usuários em tempo real
1
Criar conta gratuita
Acesse supabase.com → New Project → anote a URL e a chave anon key do projeto.
2
Criar a tabela
No SQL Editor do Supabase, cole e execute:

create table produtos (id bigint generated always as identity primary key, nome text, tipo text, coluna text, estoque int default 0, status text);
3
Habilitar acesso público
Em Authentication → Policies, adicione uma policy Allow all na tabela produtos — ou rode:

alter table produtos enable row level security; create policy "public" on produtos for all using (true) with check (true);
